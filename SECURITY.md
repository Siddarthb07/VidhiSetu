# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Email **siddarthb078@gmail.com** with:

- A description of the issue
- Steps to reproduce
- Impact assessment (if known)

We aim to acknowledge reports within 72 hours.

## Scope

This repository holds **public architecture documentation only** — not the closed-source application. Third-party dependencies in any related private tooling are tracked via Dependabot where enabled.
